# 遇见路得 · 战略方案与体系设计文档

> 本文档是实施前的规则蓝本。已锁定决策见「✅ 已定」，待定项见「⏳ 待定」。
> **核心定位：一个靠教会网络背书进门、靠认真受装备赢得曝光的盟约婚配平台。**
> 钱不是排序的主货币——「是否被真实教会接纳 + 是否愿意为婚姻受装备」才是。

## ✅ 已锁定的九条战略决策（方案 v1，2026-06 定）

| # | 决策点 | 锁定值 |
|---|--------|--------|
| 1 | 受众地理 | **海外华人基督徒**，运营不落地大陆，软件挂国外（安全考虑） |
| 2 | 宗派范围 | **改革宗 / 长老会为核心**（用「区会」长老制语汇），广义福音派可入但标注信仰背景 |
| 3 | 基督教身份 | **正大光明做基督徒平台**；保留「遇见路得」这种有美感的命名。⚠️ 旧版「清理宗教符号」决定作废 |
| 4 | 牧者 / 引荐人 | **硬门槛**——没有教会背书不给匹配。牧师与引荐人是**必填且决定性最大**的信息 |
| 5 | 真背书 | **要真背书，分两步**：MVP 阶段用户自填引荐人 + 平台人工抽查关键项；二期上自动确认（牧者收到请求、点头才算数） |
| 6 | 匹配范围 | **同宗派 / 同信仰背景优先**，跨区会可匹配、同背景加分 |
| 7 | AI 知识库 | 改革宗婚姻神学（凯勒《婚姻的意义》一路）+ 圣经辅导（CCEF 一路）+ 通用婚前辅导；**教义由牧者 / 平台审核把关** |
| 8 | 课程形式 | **读材料 + AI 出题问答 + 判定通过**的自助闯关，关键节点牧者确认 |
| 9 | 高曝光货币 | **以「完成课程 / 受装备」为主**，少量付费作为运营支持（非排序主货币） |

这套方案的一句话内核：**AI 全免费做服务层 → 课程把基督徒装备成熟 → 受装备 + 教会背书共同决定曝光与匹配优先级 → 真人红娘 / 软件特权作为少量付费的运营补充。**

## 二、信任网络与分流（平台最独特的地方）

两层分流：

### 分流一：按付费意愿（普通 / VIP）
传统的会员分级，但**钱在这个平台是次要的**——见第六节，VIP 特权弱化，曝光主要靠课程而非付费。

### 分流二：按教会归属与引荐链（核心创新）
注册时必填、决定信任分与匹配优先级：

| 字段 | 必填 | 权重 |
|---|---|---|
| **牧师 / 牧者** | ✅ 必填 | **决定性最高** |
| **引荐人 / 介绍人** | ✅ 必填 | **决定性最高** |
| 所属教会 / 区会 | ✅ | 高（同区会优先匹配） |
| 地区 | ✅ | 中 |
| 同工 / 负责人 | 选填 | 中 |
| 信仰背景 / 宗派 | ✅ | 高（同背景加分） |

> 硬门槛逻辑：没有可核实的牧者 / 引荐人，进不来、不给匹配。这一举解决婚恋平台最头疼的「骗子与安全」问题——封闭的教会信任网络天然过滤不诚信者。
> 背书两步走（决策 5）：MVP 自填 + 人工抽查 → 二期自动确认流程。架构需预留「待背书 / 已背书」状态。

### ✅ 方案 v2 新增：信仰知识测试（入池前提）

**决策**：软门槛——测试不通过可继续用平台、完善资料，但**不进匹配池**（与牧者背书逻辑一致，两道门并行）。

- **教义标准**：使徒信经 + 尼西亚信经基本范围，20 道单选题，答对 ≥15 题视为通过
- **不强推改革宗五要点**——是入门门槛，不是神学院考试
- **与课程关系**：测试是「入池前提」，课程是「曝光加分」；两者并行，互不替代
- **测试题库**：已草拟 20 题，涵盖三一论、基督论、圣灵论、救恩论、末世论、教会论、圣经观（见附录）

完整信任门槛体系：
```
注册（任何人）
  ↓
信仰知识测试（通过才进匹配池）  ← v2 新增
  ↓
牧者背书（至少 1 个才进匹配池）  ← 原有硬门槛
  ↓
进入匹配池（基础曝光）
  ↓
完成凯勒课程（曝光翻倍）         ← 原有
完成恋爱必修课（曝光加分）        ← v2 新增
牧者介绍信（信任分加权）          ← v2 新增
```

### ✅ 方案 v2 新增：牧者介绍信（信任分加权）

牧者主动为用户出具介绍信，让匹配对象「透过牧者的眼看见这个人」，深度超过个人资料自述。

**四块内容**：家庭情况 / 信仰情况 / 属灵生命状态 / 教会生活参与

**关键规则**：
- 非强制——注册后「建议填写」，有介绍信的用户信任分更高、匹配权重加分
- **可见范围：仅双方可见**（配对成功后才向对方展示）✅
- MVP 阶段：牧者直接在平台填写或通过邮件提交文字；二期做结构化表单
- 管理员可见（审核真实性），不对其他用户公开

## ✅ 方案 v3：从单页落地页 → 完整应用（2026-06 定）

这是最大的一次结构跃迁——把「遇见路得」从一个堆在单页的展示站，变成一个**有 4 种角色、有完整关系生命周期、有社群**的真实应用。

### v3-1 页面架构：混合（首页静态 + 应用区 React）
- **首页**保留现在打磨好的静态宣传页（纯宣传/介绍，引导邮箱注册）
- 登录后的**应用区用 React 挂在 `/app`**，多页：个人中心 / 社群 / 课程 / 测试 / 匹配 / 私聊 / 牧者台 / 管理台
- 完善资料 → 实时提升曝光（曝光算法真实可靠，见下）

### v3-2 曝光算法（真实可靠，资料完整度并入公式）
```
exposure = (base + 资料完整度分 + 背书加成) × 课程系数
  · base            注册基线
  · 资料完整度分     每完善一块资料 +N（驱动用户补全资料）
  · 背书加成         牧者背书 verified 后加成
  · 课程系数         完成凯勒课 ×2（翻倍）；恋爱必修课加分
  · VIP 完全不参与   钱买不到任何一项
```

### v3-3 四种角色（新增「牧者」角色）
| 角色 | 来源 | 权限 |
|---|---|---|
| 普通用户 | 邮箱注册 | 完善资料、课程、测试、AI 咨询、（达门槛后）匹配/私聊/发帖 |
| VIP | 付费 / 完课赠 | 纯便利特权（不碰曝光） |
| **牧者** | **申请 + 管理员审核升级** | 做中保、受邀背书、写推荐信 |
| 管理员 | 平台指定 | 全权 + 审核牧者认证 / 社区管理员认证 |
| **社区管理员** | **考核 + 提交认证 + 审核通过** | 管理社区：精选贴、删帖等 |

> 牧者也能作为普通用户参与匹配（单身牧者也能找对象）[v3 决策 6]

### v3-4 匹配硬门槛（三道硬门 + 一道软加分）
匹配前必须全部通过——**未达标无法进入匹配，无法私聊**：
1. **信仰知识测试**（硬，答对 ≥15/20）
2. **牧者背书**（硬，至少 1 个 verified）
3. **轻量恋爱必修课**（硬，看完 + 写感受打卡）
4. 凯勒《婚姻的意义》深度课 —— **加分不强制**（完成翻倍曝光，但不卡匹配）

> 漏斗设计：前三道门不至于太窄（恋爱课轻量、可快速完成），凯勒深度课留作高曝光激励。
> 匹配必须经算法分析优先度来匹配；匹配成功后才开放双方私聊通道。

### v3-5 关系生命周期（双方牧者审核确立）
```
单身 → 通过三道硬门槛进池 → 算法按优先度匹配 → 匹配成功开私聊
  → 想确立关系 → 联系【双方牧者】→ 双方牧者都点头
  → 确立关系（获关系徽章 + 公开「恋情确认期」）
  → 双方退出匹配池（确认期内都不会被别人匹配到）
```
- **确立关系审核 = 双方牧者都要通过**（对称、最稳妥）[v3 决策 3]
- 确立后双方匹配通道关闭，对外公开「恋情确认期」状态

### v3-6 社群（全站广场 + 区会/宗派小组）
- **结构**：一个全站广场 + 按区会 / 宗派分的小组（混合）[v3 决策 4]
- **功能**：晒帖、个人分享、个人主页（像朋友圈）
- **发帖门槛**：进了匹配池（测试 + 背书通过）才能发帖，**浏览不限**[v3 决策 5]——保证社群里都是真实信徒
- **社区管理员**：普通会员→**考核 + 向上提交认证 + 管理员审核通过**→获社区管理权限（精选贴、删帖等）[v3 用户补充]

### v3-7 个人主页（朋友圈式）
用户拥有个人主页，聚合：个人套餐 / 个人课程进度 / 个人测试状态 / 个人信息完善度 / 个人分享动态。

## 三、用户分级总览

三个等级：**普通用户（Free）· VIP · 管理员（Admin）**（v3 起增设牧者 / 社区管理员，见 v3-3）

核心原则：
- 普通用户**不坑**——核心功能全开（含完全免费的 AI 咨询、浏览、匹配理由、表达意向、参加课程）
- VIP = **纯便利**（高级筛选、谁看过我、每日更多主动次数），**完全不碰曝光排序**——曝光只认课程完成度 + 教会背书，钱买不到靠前
- **完成精品课送 14 天 VIP 体验** ✅：受装备的人不光排前面，还白拿一段便利 → 付费与受装备站到同一边，VIP 成了课程的「赠品」而非「对手」
- AI 对所有人免费；曝光主要靠课程完成度与教会背书

## 四、AI 知识库（RAG）

AI 咨询外接知识库，作为基督徒婚姻辅导的导师，**完全免费**。

- **内容来源**：改革宗婚姻神学（凯勒等）+ 圣经辅导（CCEF）+ 通用婚前辅导书籍资料
- **教义把关**：由牧者 / 平台审核，防跑偏（关键——基督徒平台不能让 AI 讲错神学）
- **技术底子**：复用现有 ChromaDB 向量库基础设施，重点在内容与把关，不是从零搭技术
- **双重用途**：① 喂 AI 咨询；② 当第四支柱「课程」的教材来源（同一套，复用最省事）

## 五、婚姻课程（曝光的主货币）

平台不只有「婚介 + 咨询」，还要有**婚姻资料页 + 婚姻课程页**。

- **课程形式**（决策 8）：读材料 → AI 出题问答 → 判定通过的自助闯关；关键节点牧者确认
- **核心机制**：**AI 辅导基督徒完成课程 → 完成者获得高曝光、高优先度、高匹配推荐**
- **意义**：把「属灵成熟度 / 受装备程度」变成曝光货币，比「花钱买曝光」正得多，也契合平台基督徒定位

### 第一门精品课（MVP 先上）

**凯勒《婚姻的意义》** — 改革宗婚姻神学经典，深度 10+ 单元，接近完整查经。

| 项目 | 规则 |
|---|---|
| 完成判定 | 读材料 + AI 问答 + **期中、结业 2 个牧者确认节点** |
| 完课奖励 | 曝光分翻倍 + 资料页「已完成婚姻装备」徽章 + **14 天 VIP 体验** ✅ |
| 积分奖励 | +300 分 ✅ |

### ✅ 方案 v2 新增：第二门课「恋爱必修课」（平行独立课程）

**主题**：基督徒约会 / 婚前预备（如何面对冲突、如何离开父母建立边界、圣经中的婚姻观）

| 项目 | 规则 |
|---|---|
| 与第一门关系 | **平行独立**，不是前置条件，可先做任意一门 |
| 完成判定 | **轻量打卡**——看完每单元 + 写 ≥50 字感受即通过，**无需牧者节点确认** |
| 完课奖励 | 曝光分提升（幅度低于凯勒课，凯勒是核心，此课是加分项） |
| 用途 | 面向未婚预备阶段，承接面最广；内容来自 AI 知识库（同一套教材复用） |

## 六、变现模型（少量付费，作运营支持，非排序主货币）

AI 全免费、曝光主要靠课程后，付费**退居次要**，只作运营现金流支持。仍分两阶段——早期不背人力成本。

### 第一阶段（现在落地，纯软件、零边际成本）
| 收费点 | 价格 ✅ | 说明 |
|---|---|---|
| **基础 VIP** | **¥29/月** | 谁看过我、更多每日主动次数等基础便利 |
| **进阶 VIP** | **¥59/月** | 含基础全部 + 价值观 / 装备程度等深度高级筛选 |
| ❌ 付费曝光 / 资料置顶 | — | **不卖**——曝光只认课程 + 背书，钱买不到排序，否则稀释根基 |
| ❌ 联系方式付费墙 | — | 不走，最坑免费用户 |

> 定价依据：对标海外华人基督徒群体支付习惯，锚定 soul / 探探等纯软件会员的行业常见水位（约 ¥29～59/月）。真正的高价（真人红娘）留到第二阶段。

### 第二阶段（模式验证后再上）
| 收费点 | 说明 |
|---|---|
| **真人红娘服务** | 专属红娘主动牵线、安排见面、破冰撮合、筛人。真有人力成本，收费天经地义。早期创始人 / 兼职按单结算（先有订单再派单，不先养人） |

## 七、积分体系（保留增长飞轮，改用途）

签到 / 邀请 / 任务这套增长机制保留，积分换**有真实价值的权益**。
⚠️ 注意：邀请拉新在「教会背书硬门槛」下要重新设计——不能开放裂变，被邀请人仍需通过引荐人 / 牧者背书才算有效。

| 类别 | 任务 | 建议积分 | 上限 |
|---|---|---|---|
| 每日签到 | 每天签到 | +10 ✅ | 当天一次 |
| 一次性 | 完善资料 | +50 | 一次性 |
| 一次性 | 完成牧者 / 引荐人背书 | +50 | 一次性 |
| 一次性 | 验证邮箱 | +20 | 一次性 |
| **课程** | **完成 1 门精品课** | **+300 ✅（主要积分来源）** | 每门一次 |
| 站内 | 完整看完 1 人资料并发意向 | +10 | 一次/天封顶 |

兑换项（建议）：**VIP 体验天数**（高级筛选 / 谁看过我）、单次红娘咨询（二阶段）、课程相关周边。
**兑换比例：100 分 = 1 天 VIP 体验 ✅**（攒 10 天签到换 1 天，让 VIP 体验显贵重）。
> ⚠️ 积分**不能换曝光 / 置顶**——曝光只认课程 + 背书，积分换便利可以，换排序不行（与第六节一致）。
> 数值锚点：签到 +10/天、完课 +300（≈1 个月签到，受装备明显比刷脸值钱）、100 分换 1 天 VIP。全部锁定 ✅。
